# Invio di una richiesta ad Adobe{#submitting-request-to-adobe}

>[!NOTE]
>
>Per poter continuare, accertatevi che IMS sia attivato sul sistema. Consultate la sezione su [IMS](../../integrations/using/about-adobe-id.md).

Una volta abilitato IMS, puoi richiedere il provisioning dell&#39;integrazione del servizio di base Persone/ Audience Manager con Campaign. A tal fine, inviate un messaggio e-mail a [Digital-Request@adobe.com](mailto:Digital-Request@adobe.com) con le seguenti informazioni:

  </tr> 
  <tr> 
   <td> <strong>Nome organizzazione:</strong><br /> </td> 
   <td> Nome organizzazione </td> 
  </tr> 
  <tr> 
   <td> <strong>ID organizzazione IMS</strong><br /> </td> 
   <td> L’ID organizzazione IMS. <br> Puoi trovare il tuo ID organizzazione IMS nel Experience Cloud , nel menu Amministrazione. Viene fornito anche quando si effettua la prima connessione all’Adobe Experience Cloud. </td> 
  </tr> 
  <tr> 
   <td> <strong>Ambiente:</strong><br /> </td> 
   <td> Esempio: Produzione </td> 
  </tr>
